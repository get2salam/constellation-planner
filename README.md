# Constellation Planner

A local-first strategy map where projects become stars and dependencies become visible paths.

Constellation Planner is for founders, operators, and solo builders who want more than a flat task list. Plot your bets on a living star map, score their upside, and see which ideas connect to your bigger mission.

## Planned features

- Interactive constellation canvas
- Strategy nodes with impact, confidence, and effort
- Dependency links between projects
- Ranked priority insights
- Local-first persistence
- Import and export backups

## Quick start

```bash
git clone https://github.com/<you>/constellation-planner.git
cd constellation-planner
python -m http.server 8000
```

Then open <http://localhost:8000>.
